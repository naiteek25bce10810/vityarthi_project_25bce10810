# vityarthi_project_25bce10810
Project title:  Centralized Digital Health Record System
Overview: This project solves a real word problem of carrying big files and papers whenever we visit doctor or the hospitals. This project turns everything online make it simpler for the doctor as well as the patient
Technology used: This project is coded in Python language, with the help of 3 libraries i.e csv, os, datetime. Also using built in functions and self made functions.
Instruction to use: As such no instruction but if you use the code in Visual Studio Code it would be rather better because Visual studio code supports all the libraries of the python
